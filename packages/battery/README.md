# @danyalwe/capacitor-battery

Capacitor plugin for battery monitoring

## Install

To use npm

```bash
npm install @danyalwe/capacitor-battery
````

To use yarn

```bash
yarn add @danyalwe/capacitor-battery
```

Sync native files

```bash
npx cap sync
```
