# @danyalwe/capacitor-systeminfo

Capacitor plugin for device system information

## Install

To use npm

```bash
npm install @danyalwe/capacitor-systeminfo
````

To use yarn

```bash
yarn add @danyalwe/capacitor-systeminfo
```

Sync native files

```bash
npx cap sync
```
